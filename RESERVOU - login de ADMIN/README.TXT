Essa é a página de login de ADMINISTRADORES para a aplicação web Reservou.


1. Como rodar o site:
    Para rodar, apenas dê o comando "npm start" no terminal, e então abra no navegador o link http://localhost:3000/. 

2. Testando as funcionalidades: 
    - Para testar o login, pode-se usar o email "admin@admin.com.br" e a senha "admin", assim, deve aparecer a mensagem que o login foi realizado com sucesso.
    - Caso a senha esteja errada, deve aparecer uma mensagem acusando que a senha está incorreta.
    - Caso o email não exista, deve aparecer uma mensagem de que o email não foi encontrado.
    - O botão da logo ainda não funciona como deveria, pois não se tem a página home para ser anexada.
